Chatbot SESI/SENAI de TI

Este é um chatbot simples, desenvolvido como parte de um projeto do SESI/SENAI, focado em responder perguntas sobre uma ampla gama de tópicos de Tecnologia da Informação (TI).
Tecnologias Utilizadas

O projeto foi desenvolvido inteiramente em Python, utilizando a biblioteca tkinter para a criação da interface gráfica do usuário.

    Python: Linguagem de programação principal utilizada para toda a lógica do chatbot e manipulação da interface.

    Tkinter: Biblioteca padrão do Python para criar interfaces gráficas (GUIs). Utilizada para construir a janela de login, a tela de chat, campos de entrada, botões e exibir as conversas.

    tkinter.messagebox: Módulo do Tkinter para exibir caixas de diálogo de mensagens, como alertas de erro no login.

    tkinter.scrolledtext: Um widget de texto com barra de rolagem embutida, ideal para a área de conversação do chat, permitindo que o histórico de mensagens seja visualizado e rolado automaticamente.

Funcionalidades

    Tela de Login: Autenticação simples com usuário e senha predefinidos para acessar o chatbot.

    Chat Interativo: Interface para enviar perguntas e receber respostas do chatbot.

    Ampla Base de Conhecimento em TI: O chatbot pode responder sobre:

        Linguagens de Programação (Python, Java, JavaScript, C#, C++, PHP, Swift, Kotlin, Rust, Go)

        Desenvolvimento Web (HTML, CSS, React, Angular, Vue.js, Node.js, Frontend, Backend)

        Bancos de Dados (SQL, MySQL, PostgreSQL, MongoDB, NoSQL)

        Cloud & DevOps (AWS, Azure, GCP, Docker, Kubernetes, Git, DevOps)

        Redes e Segurança (IP, DNS, HTTP/HTTPS, Firewall, VPN)

        Conceitos Gerais de TI (API, IDE, VS Code, POO, Framework, Biblioteca, Bug)

        Hardware e Sistemas Operacionais (Linux, Windows, macOS, Android, iOS, CPU, RAM, SSD, HD)

    Respostas Conversacionais: Capacidade de interagir com "small talk" (saudações, perguntas sobre o próprio bot, piadas de TI).

    Temas Dinâmicos: Alternância entre tema claro e tema escuro para a interface do aplicativo.

    Limpar Conversa: Botão para apagar todo o histórico de mensagens do chat.
